<h1 align="center">A World time clock built upon React🌍🕔</h1>

## Usage

### Use bash

```bash
$ git clone git@github.com:thalesf/react-world-clock.git
$ cd react-world-clock
$ npm install or yarn
$ npm start or yarn start
# visit http://localhost:3000
```
